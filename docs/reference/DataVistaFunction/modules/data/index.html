Data Table

This component is used to display historical data for selected points within a specific time range.

Features​

PropertyTypeDescription
TagsTag ArrayList of points associated with the component

Settings​

Click the input box to open the point association window and set associated points for the component. Click the "Add Option" button to add associated points.

Table Features​

PropertyTypeDescription
font color (header)ColorText color of the table header
bg color (header)ColorBackground color of the table header
font color (row)ColorText color of the table content
bg color (row)ColorColor Background color of the table row
grid color (row)ColorBorder color of the table row

Table Parameter Settings​

Table Column Description​

PropertyTypeDescription
Data TimetimeTime when the data for the collection point was recorded
Other ColumnsTagValues corresponding to the selected points at the given time

Filtering​

You can filter the displayed data for a specific time range using the date range option in the top right corner of the component.
