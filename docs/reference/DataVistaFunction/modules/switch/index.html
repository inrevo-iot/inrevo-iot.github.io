Switch

This component allows you to create a switch that toggles between "On" and "Off" states. It also supports defining change events.

Features​

PropertyTypeDescription
defaultColorColorThe default color of the switch
activeColorColorThe color when the switch is in the "On" state

Configuration​

For additional features, please refer to the Common Configuration.
