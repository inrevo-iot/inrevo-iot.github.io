RealTime

This component is used to display real-time data for points in a table format.

Features​

PropertyTypeDescription
TagsTag ArrayList of points associated with the component

Settings​

Click the input box to open the point association window and set associated points for the component. Click the "Add Option" button to add points.

Table Features​

PropertyTypeDescription
font color (header)ColorText color of the table header
bg color (header)ColorBackground color of the table header

bg color (row)ColorBackground color of the table row
grid color (row)ColorBorder color of the table row
