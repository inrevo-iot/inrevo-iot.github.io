文本域

该组件允许您创建一个文本域组件，且支持您定义组件的变更事件。

特征​

属性类型描述
contentText支持设置默认的展示文本，且支持根据点位数据配置默认文本

配置​

关联动态点位数据​

点击点位ICON , 打开点位关联窗口，对组件设置关联点位，并设置对应文本内容。

其他特征请参见组件共有配置。
