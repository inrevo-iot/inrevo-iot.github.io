视图

创建视图​

展开项目，您可以看到一个“创建视图”的按钮，点击“创建视图”，您将看到一个创建视图的弹窗：

填入视图名称，点击“保存”按钮，您就创建成功了一个视图。

编辑视图​

选中视图，你将看到下图所示的界面。

编号①展示所在位置是项目的视图列表

编号②是视图的编辑按钮，点击编号②，将进入编辑模式

编号③是视图的分享功能，被分享的页面可以直接访问。分享也可以被取消，被取消的分享将无法访问。

编号④是全屏的操作图标，点击图标可以切换成网页内全屏展示。

点击编号⑤，可以切换左侧侧边栏的展示。

点击编号②，进入编辑模式

编号描述
1将选中的组件的层级至于顶部
2将选中的组件的层级至于底部
3将选中的组件的层级上移一层
4将选中的组件的层级下移一层
5将选中的组件左对齐
6将选中的组件右对齐
7将选中的组件上对齐
8将选中的组件下对齐
9撤销操作
10还原操作
11组件的tab，该tab下展示了页面可添加的组件，点击下方的组件图标，可直接将组件添加到页面上。
12视图属性或组件属性的编辑tab。如果视图中无组件选中，显示的是视图的属性编辑模块；如果视图中有组件选中，显示的是组件的属性编辑模块。
13右侧窗口收起或展示的状态切换图标。
14关闭按钮，点击将收起右侧配置窗口。

快捷键支持​

快捷键描述
Ctrl + C复制选中的组件
Ctrl + V粘贴选中的组件
Delete删除选中的组件

鼠标操作支持​

快捷键描述
位移鼠标放到组件上，出现 图标时，可拖动组件
调整大小鼠标放到组件周围的边和角上的点时，可以对组件区域进行放大或缩小
旋转鼠标放到组件右上角的图标上时，可对进行旋转

删除视图​

点击视图列表右侧的删除图标，可以对视图进行删除操作。
